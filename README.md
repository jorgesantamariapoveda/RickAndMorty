### Entorno
- MacBook Air M1 (8 GB, 512 GB SSD)
- macOS Sonoma 14.2.1
- Xcode 15.3
- iOS 16.0
- Swift 5.10

### Decisiones adoptadas
- SwiftUI + Async await
- Principios SOLID + clean code.
- Arquitectura MVVM + clean arquitecture.
- No usar librería de terceros.
